# Network Security

## About

Network Security is the practice of protecting computer networks, connected devices, and transmitted data from unauthorized access, attacks, misuse, and disruption.

It combines hardware, software, security policies, and best practices to ensure secure communication.

---

# Objectives of Network Security

- Protect Confidential Data
- Prevent Unauthorized Access
- Ensure Data Integrity
- Maintain Network Availability
- Detect and Respond to Threats

---

# Importance of Network Security

Network Security helps to:

- Prevent Data Breaches
- Protect Business Information
- Secure Online Transactions
- Reduce Cyber Attacks
- Ensure Business Continuity

---

# Types of Network Security

## 1. Access Control

Access Control allows only authorized users and devices to access network resources.

Examples

- User Authentication
- Role-Based Access Control (RBAC)
- Access Control Lists (ACLs)

---

## 2. Firewall

A Firewall monitors and filters incoming and outgoing network traffic based on predefined security rules.

Types

- Packet Filtering Firewall
- Stateful Firewall
- Proxy Firewall
- Next-Generation Firewall (NGFW)

---

## 3. Intrusion Detection System (IDS)

An IDS monitors network traffic and alerts administrators about suspicious activities.

Types

- Network IDS (NIDS)
- Host IDS (HIDS)

---

## 4. Intrusion Prevention System (IPS)

An IPS detects and automatically blocks malicious network traffic.

Functions

- Detect Threats
- Block Attacks
- Generate Alerts

---

## 5. Virtual Private Network (VPN)

A VPN creates an encrypted tunnel between a user and a network.

Benefits

- Secure Remote Access
- Protects Data on Public Wi-Fi
- Encrypts Internet Traffic

---

## 6. Network Segmentation

Network Segmentation divides a network into smaller sections to improve performance and security.

Examples

- VLANs
- Separate Guest Network
- Department-Based Networks

---

## 7. Wireless Security

Protects Wi-Fi networks from unauthorized access.

Best Practices

- WPA3 Encryption
- Strong Wi-Fi Password
- Disable WPS
- Hidden SSID (optional)

---

## 8. Endpoint Security

Protects devices connected to the network.

Examples

- Antivirus
- Endpoint Detection and Response (EDR)
- Device Encryption

# Real-Life Example

A company allows employees to work remotely.

To secure the connection:

- Employees connect using a VPN.
- The firewall filters incoming traffic.
- IDS monitors suspicious activity.
- MFA is required before network access.

This combination significantly improves the organization's network security.
